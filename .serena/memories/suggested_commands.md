# Suggested Commands

## Development
- `wails dev`: Start the application in development mode with hot-reloading.
- `wails build`: Build a production-ready binary.
- `go mod download`: Download and install Go dependencies.
- `cd frontend && npm install`: Install frontend dependencies.

## Testing
- `go test ./...`: Run all backend (Go) unit tests.
- `go test -v ./internal/core/...`: Run core logic tests with verbose output.

## Maintenance
- `go mod tidy`: Remove unused Go dependencies.
- `go fmt ./...`: Format Go code.
